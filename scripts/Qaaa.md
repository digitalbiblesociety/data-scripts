This code represents the start of the range of ISO 15924 codes (Qaaa–Qabx) reserved for private use. Scripts assigned codes in this range are not standardised by the Unicode Consortium or ISO and their use is determined entirely by private agreement between the parties using them. They may represent invented scripts, scripts under development, or scripts used for experimental or internal purposes. No standard rendering behaviour or character properties are defined for private-use script codes.
