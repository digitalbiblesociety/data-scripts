The ISO 15924 code *Zsym* is used to tag text that consists of symbols not associated with any particular writing script. This includes mathematical symbols, pictographs, emoji, currency signs, and other non-alphabetic symbol sets. In Unicode, such characters are assigned to the *Symbol* script category. The code is used primarily for metadata and language-tagging purposes and does not correspond to a single encoding or script block in Unicode; symbols are spread across many different Unicode blocks including Mathematical Operators, Miscellaneous Symbols, Dingbats, Emoji, and others.
