The Old Uighur language is an extinct variety of Old Turkic spoken in the Tarim basin. It is an ancestor of the modern Uyghur language. The language was written using what has been described as an “alphabetized abjad”, with matres lectionis used for indicating both long and short vowels. The Old Uighur script was the prototype for the Mongolian and Manchu scripts.
